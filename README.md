# BillScraper.js
A JavaScript API to scrape and analyze bills in Congress.
  * Returns full Congressional bill text and metadata from Congress.gov as JSON.
  * Parses bills for key points and ideas.
  * Powerful filtering.

## Table of Contents

* `test`: Holds all unit tests.
